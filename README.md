# Dumbo Octopus Webpage

This is a simple HTML and CSS webpage about the Dumbo Octopus, a fascinating deep-sea creature known for its ear-like fins. 
The website provides interesting facts about the animal and includes images arranged using responsive CSS styling.

## Features

* Informative content about the Dumbo Octopus
* Three images, including a centered featured image
* Responsive layout using CSS Flexbox
* Clean and accessible HTML structure
* Custom styling with CSS

## Technologies Used

* HTML5
* CSS3

This project was created as part of a web development exercise to practice HTML, CSS, image positioning, lists, 
and responsive design principles.
